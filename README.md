# **Website Traffic & Engagement Analysis using Python**
# Overview

This project analyzes website traffic and user engagement metrics from a CSV dataset exported from an analytics tool. The goal was to uncover patterns in user sessions, identify top-performing marketing channels, and explore engagement behavior over time.

# Objectives

Identify trends in website sessions and users over time

Compare marketing channels based on user acquisition and engagement

Analyze which channels drive the most active and engaged users

Understand engagement rate variation across channels

# Tools & Libraries

Python

Pandas – data cleaning & transformation

NumPy – numerical operations

Matplotlib / Seaborn – data visualization

# Analysis Performed

Time series visualization of sessions and users

Channel-wise comparison of user count and engagement time

Engagement rate distribution by channel

Relationship between engagement rate and sessions over time

# Visualizations

Line chart: Sessions and Users over Time

Bar chart: Total Users by Channel

Bar chart: Average Engagement Time by Channel

Box plot: Engagement Rate Distribution

Line chart: Engagement Rate vs Sessions over Time
